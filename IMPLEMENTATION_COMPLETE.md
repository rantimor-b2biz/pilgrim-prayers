## What This Enables

### ✅ Clear Collaboration
- UX knows exactly what Paid Search owns (keywords, targeting, ad copy)
- Paid Search knows exactly what UX owns (landing pages, forms, speed, testing)
- Both know when to communicate and how

### ✅ Structured Syncs
- **Monday:** UX briefs PS on friction points (5 min read)
- **Wednesday:** PS briefs UX on traffic quality (5 min read)
- **Friday:** Both review together and decide next week's top 3 (30 min)
- Templates ensure consistency week-to-week

### ✅ Data Sharing
- UX shares: Clarity heatmaps, funnel metrics, A/B results, mobile analysis
- PS shares: Search terms, traffic volume, conversion rates, quality scores
- Both use the same KPIs (conversion rate, CPA, page load time, mobile CR, return visitors)

### ✅ Fast Escalation
- Know immediately when to alert each other:
  - Conversion drops >10%?
  - Page breaks?
  - Mobile converts 2x+ desktop?
  - Quality score tanks?
  - New keyword emerges?

### ✅ Standard Templates
- Both use templates from their skill files
- Consistency across reports
- Easy to scan and understand
- Less time explaining, more time optimizing

### ✅ Quarterly Alignment
- Deep dive on strategy
- Update memory files with learnings
- Adjust priorities based on data
- Confirm direction is still correct

## When to Escalate Immediately

Alert each other right away (not waiting for next sync) if:

- ✋ Conversion drops >10%
- ✋ Conversion drops >20% (critical)
- ✋ Landing page breaks
- ✋ Mobile converts 2x+ desktop
- ✋ Quality Score drops below 5
- ✋ Traffic spikes 40%+ but conversion drops
- ✋ New emerging keyword with high intent
- ✋ Form abandonment spikes suddenly
