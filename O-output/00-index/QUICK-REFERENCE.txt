================================================================================
FILE NAMING
================================================================================

[DATE]-[TYPE]-[SUBJECT].md

Examples:
  2026-02-24-week8-garden-tomb-brief.md
  2026-02-24-garden-tomb-research.md
  2026-02-24-garden-tomb-post.md
  2026-02-24-garden-tomb-email.md

Breakdown:
  [DATE]     = When file was created (YYYY-MM-DD)
  [TYPE]     = What kind of file (week8, garden-tomb, email, etc)
  [SUBJECT]  = What it contains (brief, research, post, email)
