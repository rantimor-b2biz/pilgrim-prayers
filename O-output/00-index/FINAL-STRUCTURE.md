## 🎯 What's Where

### **Navigation (00-index/)**
Everything you need to understand the structure:

- `STRUCTURE.md` — Visual diagrams
- `REORGANIZATION-SUMMARY.md` — What changed

### **Planning (01-briefs/)**
All content planning documents before production:

  - Quick start guide

### **Production (02-deliverables/)**

## 🚀 Getting Started With New Structure

### **For Content Teams**
1. Read: `00-index/INDEX.md`
2. Plan week: Save briefs to `01-briefs/week-[X]/`
3. Produce: Agents save to `02-deliverables/week-[X]-deliverables/[agent]/`
4. Publish: Move to `03-published/blog-posts/2026-02/week-[X]/`

### **For Project Leads**
1. Reference: `00-index/STRUCTURE.md`
2. Track: Check `02-deliverables/` to see progress
3. Publish: Move completed work to `03-published/`
4. Archive: Everything stays for reference

### **For Marketers**
1. Find campaigns: `04-campaigns/email/2026-02/`
2. Find ads: `06-ads/search-campaigns/`
3. Historical data: `05-archive/`
