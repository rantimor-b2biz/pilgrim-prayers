# Twitter/X Thread Skill

Create threads that people can't stop reading.

## When to Use

- Writing Twitter/X threads
- Converting long content into thread format
- Creating educational or thought leadership threads

---

## Thread Basics

**Why threads work:**
- 10x more engagement than single tweets
- Algorithm favors them (more time on platform)
- Showcase expertise without leaving Twitter
- Highly shareable and quotable

**Length sweet spot:** 7-12 tweets
- Too short (3-5): Feels incomplete
- Too long (15+): Reader fatigue
- Perfect (7-12): Enough depth, respects attention

---

## Thread Structure

Every great thread follows this arc:

```
ACT 1: HOOK (Tweet 1-2)
Stop the scroll
Create curiosity gap
Make a promise

ACT 2: DELIVER (Tweet 3-10)
Teach the concept
Share the insight
Build the case
Give examples

ACT 3: CLOSE (Tweet 11-12)
Recap value
Call to action
Drive next step
```

---

## Hook Formulas (Tweet 1)

The hook has ONE job: Make them read tweet 2.

### Formula 1: Bold Declarative
Make a strong, unexpected claim.

```
You're not failing because of lack of discipline.

You're failing because the friction is invisible.
```

### Formula 2: Pattern Observation
```
A pattern I see in people who succeed at building habits:

They don't look for motivation.

(Here's what they look for instead)
```

### Formula 3: Surprising Number
```
Wendy Wood tracked 20,000 people for 10 years.

Found that one small thing led to 3x more success.

Not motivation. Not willpower. Something much simpler.
```

### Formula 4: Personal Story
```
I tried to run every morning for 6 months.

Zero days.

Then I read one study that changed everything.
```

### Formula 5: Contrarian Challenge
```
Everyone tells you to "just start!"

That's why no one starts.

Motivation isn't the problem. (Thread)
```

## Tweet-by-Tweet Rules

### Rule 1: One Idea Per Tweet
Each tweet = one clear thought.

**Bad:**
```
The problem is friction which means small obstacles that make habits harder like shoes in the closet instead of by the bed or a book in your bag instead of on your desk.
```

**Good:**
```
The problem is friction.

Friction = small obstacles you don't see.

Shoes in the closet instead of by the bed.
Book in your bag instead of on your desk.
```

### Rule 2: Vary Tweet Length
Create rhythm. Mix short punches with longer explanations.

```
Tweet 1: Short (1 line)
Tweet 2: Medium (2-3 lines)
Tweet 3: Short (1 line)
Tweet 4: Medium (2-3 lines)
```

### Rule 3: Use Line Breaks
Don't write paragraphs. Write lines.

**Bad:**
```
The researcher did an experiment where she asked people to put their shoes by the bed and the results were surprising because that group exercised 3x more than the group that got a motivation lecture.
```

**Good:**
```
The researcher did an experiment:

Group A: Motivation lecture
Group B: Put shoes by the bed

Results?

Group B exercised 3x more.
```

### Rule 4: Build Tension → Release
Create mini-cliffhangers throughout.

```
Tweet 5: The results were surprising. (TENSION)

Tweet 6: Group B exercised 3x more than Group A. (RELEASE)

Tweet 7: But here's where it gets interesting... (NEW TENSION)
```

## Closing Your Thread (Last 2 Tweets)

### Option 1: Recap + CTA
```
Tweet 11:
Summary:
• [Key point 1]
• [Key point 2]
• [Key point 3]

Tweet 12:
If you found this helpful:
1. Follow for more like this
2. Retweet the first tweet to share
```

### Option 2: Question + Engagement
```
Tweet 11:
So what's your invisible friction?

The thing that seems trivial—
is probably what's stopping you.

Tweet 12:
Share in the replies.

Happy to help you identify it.
```

### Option 3: Actionable Takeaway
```
Tweet 11:
Here's what to do:

1. Pick one habit you want to build
2. Identify the small friction
3. Remove it today

Tweet 12:
Small changes. Big results.

Follow for more threads like this.
```

## Thread Types

### 1. Research Breakdown (8-10 tweets)
Hook → Who did research → What they found → Why it matters → What to do

### 2. Story → Insight (7-9 tweets)
Hook → Personal story → Turning point → The lesson → Application

### 3. Tactical Framework (10-12 tweets)
Hook → Problem → Framework intro → Step 1-3 → Example → Recap

### 4. Contrarian Take (6-8 tweets)
Bold claim → What everyone says → Why they're wrong → Better approach

### 5. List-Based (7-10 tweets)
"X things I learned about Y" → Item 1-5 → Pattern → Takeaway

## Quality Checklist

Before publishing:

- [ ] Hook would make YOU stop scrolling?
- [ ] Each tweet has one idea only?
- [ ] Rhythm varies (short/medium tweets)?
- [ ] Line breaks make it readable?
- [ ] Thread can stand alone (no external context needed)?
- [ ] Clear CTA at the end?
- [ ] 7-12 tweets total?
- [ ] No link in first tweet?

---

## Common Mistakes

1. **Weak hook** — Fix: Use the formulas, be bold
2. **Too academic** — Fix: Add personality, use "you"
3. **No rhythm** — Fix: Vary tweet length
4. **Vague takeaways** — Fix: Be specific ("put shoes by bed" not "plan ahead")
5. **Buried value** — Fix: Make key insights standalone tweets

---

## Pro Tips

1. **Post timing:** Test what works for your audience
2. **First 30 minutes matter:** Engage with every reply
3. **Quote RT later:** Share again a few hours after posting
4. **No links in tweet 1:** Twitter suppresses reach on links
