# LinkedIn Post Skill

Create posts that stop the scroll and start conversations.

## When to Use

- Writing LinkedIn posts
- Converting ideas into LinkedIn format
- Creating thought leadership content

---

## LinkedIn Basics

**What works on LinkedIn:**
- Personal stories with professional insights
- Contrarian takes with evidence
- Practical frameworks people can use
- Vulnerability that teaches

**Post length sweet spot:** 150-300 words
- Too short: Doesn't provide enough value
- Too long: Loses attention
- Perfect: Enough depth, respects time

---

## The Hook (First 2 Lines)

The hook is EVERYTHING. Most people only see the first 2 lines before "...see more"

### Hook Formula 1: Bold Statement
```
I stopped doing [common thing].

Here's what happened:
```

### Hook Formula 2: Counterintuitive Claim
```
The worst career advice I ever got:

"[Common advice everyone gives]"
```

### Hook Formula 3: Specific Story Start
```
Last Tuesday, I got a message that changed how I think about [topic].
```

### Hook Formula 4: Surprising Number
```
I spent 20 hours/week on [task].

Now I spend 7.

Here's the shift:
```

### Hook Formula 5: Confession
```
I used to believe [common belief].

I was wrong.
```

## Post Structures That Work

### Structure 1: Story → Lesson
```
[Hook: Start in the middle of action]

[Context: Quick setup]

[Conflict: What went wrong/the challenge]

[Resolution: What happened]

[Lesson: The insight others can apply]

[CTA: Question or call to engage]
```

### Structure 2: Before/After
```
Before: [Old state - pain point]
- [Specific detail]
- [Specific detail]
- [Specific detail]

After: [New state - transformation]
- [Specific result]
- [Specific result]
- [Specific result]

The shift? [One key insight]
```

### Structure 3: Framework/List
```
[Hook about the topic]

Here's the [X]-step framework I use:

1. [Step] — [Brief explanation]
2. [Step] — [Brief explanation]
3. [Step] — [Brief explanation]

[Closing insight]

[CTA]
```

### Structure 4: Contrarian Take
```
Everyone says [common advice].

I disagree.

Here's why:

[Point 1]
[Point 2]
[Point 3]

What actually works:

[Your alternative]
```

## Writing Rules

### Rule 1: One Idea Per Post
Don't try to say everything. One clear message, well delivered.

### Rule 2: Short Lines
```
Not this:
"I realized that after years of trying to optimize my productivity by working longer hours, the real answer was actually working fewer hours but with more focus."

This:
I worked longer hours.
Productivity went down.

I worked fewer hours with more focus.
Everything changed.
```

### Rule 3: White Space
Break up your text.

Single sentences get attention.

Paragraphs get skimmed.

### Rule 4: Specificity
```
Bad: "I improved my results significantly"
Good: "My close rate went from 12% to 34%"
```

### Rule 5: End With Engagement
Ask a question. Invite a response. Make it easy to comment.

```
Good endings:
- "What would you add to this list?"
- "Have you experienced this? Share below."
- "Agree or disagree?"
- "What's worked for you?"
```

---

## What to Avoid

1. **Starting with "I'm excited to announce..."** — No one cares about your excitement
2. **Walls of text** — Break it up
3. **Vague claims** — "Best" means nothing without proof
4. **Multiple CTAs** — One ask per post
5. **Links in the post body** — Put them in comments (better reach)
6. **Hashtag stuffing** — 3-5 max, or none

---

## Engagement Tactics

### Boost Your Reach
1. Respond to every comment in the first hour
2. Comment on others' posts before and after publishing yours
3. Post consistently at the same time
4. Put links in the first comment, not the post

### Best Times to Post
- Weekday mornings (7-9 AM your audience's timezone)
- Lunch hours (12-1 PM)
- Test and track what works for YOUR audience

## Quality Checklist

Before publishing:

- [ ] Hook would make YOU stop scrolling?
- [ ] One clear idea/message?
- [ ] Short lines and white space?
- [ ] Specific details (numbers, names, dates)?
- [ ] Sounds like you (not corporate)?
- [ ] Ends with engagement opportunity?
- [ ] No link in body (moved to comments)?
- [ ] 3 or fewer hashtags?
